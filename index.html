Mục đích

Web Truyện Hay được tạo để chia sẻ truyện chữ miễn phí, phục vụ bạn đọc yêu thích truyện ngắn, truyện dài và light novel. Nội dung trên đây là demo — bạn có thể thay bằng truyện thật sau khi fork repo.

Hướng dẫn nhanh

  1. Fork repo này vào tài khoản của bạn.
  2. Chỉnh sửa index.html và style.css theo nội dung bạn muốn.
  3. Commit và bật GitHub Pages: Settings → Pages → Branch: main → Folder: / (root).
  4. Truy cập: https://tenban.github.io/truyenhay

Muốn deploy nâng cao? Xem README gốc để dùng Codespaces + Fastly Compute.

Xem hướng dẫn chi tiết

Hướng dẫn triển khai (tóm tắt)

  1. Đảm bảo có index.html trong thư mục gốc của repo.
  2. Vào Settings → Pages → chọn Branch = main và Folder = / (root) → Save.
  3. Đợi vài phút → mở link GitHub Pages của bạn để kiểm tra.

Muốn dùng Fastly Compute giống template gốc: fork repo gốc, mở Codespace và làm theo README để thêm biến môi trường FASTLY_API_TOKEN rồi nhấn Publish.
